## üé® Contrast Ratio Calculator

To verify any color combination:
```
Contrast Ratio = (L1 + 0.05) / (L2 + 0.05)

Where:
- L1 = relative luminance of lighter color
- L2 = relative luminance of darker color
- Luminance = 0.2126*R + 0.7152*G + 0.0722*B (for sRGB)
```
